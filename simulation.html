📊 Calendar Time Component Simulation

Concept

This simulation models the relationship between execution time (CPU time) and real-world calendar time considering resource constraints.

Key Equation

dt/dτ = (θ + μr λ) / (P ρ)

  • θ → Resource usage per CPU hour
  • μr → Resource usage per failure
  • λ → Failure intensity (failures / CPU hr)
  • P → Total resources available
  • ρ → Utilization (0 to 1)
